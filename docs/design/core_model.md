# SchemaMem 核心心智模型

> 面向 AAAI-27。本文档是论文引言/动机/方法的思想底座,四段式:领域现状与不足 → 人类记忆如何做到 → 我们的动机 → 核心心智模型。每个断言均已按一手来源核实(见括注);"我们的综合"处已诚实标注,不冒充既有结论。

## 一、领域现状与不足:三个被分头研究的失败,是同一个病根

### 1.1 记忆演化被切成三个互不通气的子问题

一个长期运行的智能体,对同一个实体(用户、项目、世界状态)会反复收到彼此有张力的证据。如何让记忆随之演化,当前领域(据 Hu 等 2026 综述的组织方式)分成三条独立的研究线,各自有一套默认判据:

- **更新(Updating)**:面对矛盾,默认**最新即胜**——新值覆写或使旧值失效。综述把它框成 stability–plasticity dilemma,原话:决定"何时覆写既有知识、何时把新信息当作噪声"。
- **巩固(Consolidation)**:面对相似,默认**相似即并**——通过反思式整合把片段抽象成高层要点。综述把它定义为"合并新旧记忆、形成更泛化的洞见",目的是"让学习累积而非孤立(cumulative rather than isolated)";并明确点出其代价:"有信息平滑的风险,离群事件或独特例外会在抽象中丢失"。**注意:综述把 isolated 当成要消除的东西——而我们主张有一类 isolated(例外)恰恰必须被保护、不能被巩固掉。**
- **遗忘(Forgetting)**:面对存储压力,默认**久不用即删**——按时间/频率/重要性回收。综述亦点名其失败:"像 LRU 这样的启发式会删掉长尾知识——很少被访问但对正确决策至关重要"。

### 1.2 三条线在同一类输入上同时崩溃——共同的根源是"缺期望"

考虑这样一条观测:**罕见、携带信息、不复现、且与当前信念矛盾**——比如一个自述"不吃肉"的用户,某一次点了鸡肉三明治(生日聚餐)。

- 更新说:覆写——**把一次例外误当成饮食改变**。
- 巩固说:揉进去——**把这条离群点平滑掉**。
- 遗忘说:删掉——**它罕见、久不再现,正好被回收**。

三个默认判据**全错**,且错的方向不同。正确动作是第四种:**单独留着它,当作例外,信念一动不动。**

**为什么三支同时错**:因为**三支都不携带"期望"**,所以它们只能用相似度、新近度、时间与频率这些**表面信号**去替代那个真问题——"这条观测,相对我当前所信,究竟意味着什么"。表面信号是可算的量,但它们回答的是别的问题:相似度回答"这条像不像已有的",新近度回答"这条是不是新的",时间/频率回答"这条最近被访问过几次"。而记忆演化的真问题是"这条**违反**了我此刻应有的期望到什么程度"——一个只有当**存在期望**时才可算的量(**残差**)。表面信号既没能力算它,也没意识到该算它。

于是当前**没有任何一个系统把"例外"作为一个显式的记忆结局**——因为要辨认出"孤立的强违反",先得能算"违反度",而算违反度先得有"期望"。缺的不只是例外这一类,更根本的是**判据**:一个由生成模型给出期望、把"违反度"作为唯一货币的判据。

> **必须诚实标注的边界**:综述在三个不同章节各自命名了这三个损失,但**它自己没有把三者连成"同一个病根"**。"三支共缺一个'携带期望的判据',因而只能用表面信号"是**我们的综合**,可作引言的核心 insight,但不可写成"综述指出"。

### 1.3 这不是"没人做过冲突处理",而是"做的都缺一个类别"(竞品核查结论)

必须公允承认领域已有的推进,并精确划出它们停在哪:

- **时序知识图谱(Zep/Graphiti)**:用双时间戳给旧边打 `invalid_at` 而非物理删除——解决了"能不能查到旧值",但遇矛盾仍**默认新值取代旧值**,不问"这是改变还是例外"。
- **PREMem(EMNLP'25 Findings)**:明用图式理论(Piaget/Bartlett),背景段甚至提到"exception handling"——但**例外仅是认知背景引用,未实现**;其真实更新规则是"新簇与旧片段语义匹配即删旧值",无累积、无推迟、无例外分支。它做的是跨会话推理综合,不是改变/例外仲裁。PREMem 自己承认现有系统"追踪偏好变化、消解矛盾仍吃力"——它点了名却没解决,是完美的 **foil**。
- **EM-LLM(ICLR'25)**:属长上下文/推理时方法,用 surprise 切 token 事件、扩展单次推理的上下文,**无实体、无信念、无跨会话持久化**。属综述明确排除于 agent memory 之外的类别,与本问题不在同一空间。

**基准侧的旁证(非方法对手)**:标准冲突基准 MAB FactConsolidation 上现有系统整体表现远低于饱和——单跳档次尚未超出中等水平,多跳档次几乎全线倒地(具体分数投稿前需以综述/原论文原文重核后填入,不在此文档硬编)。这说明"改变"这条轴在公认基准上仍有巨大 headroom。而新出的评测框架 LoCoMo-Plus(2026-02)在测"冲突识别"的同时,其 Limitations **明确声明不建模 long-term belief revision**——这从评测侧佐证"改变 vs 例外的信念修订"这条轴重要、且仍空着。(二者均为基准,用作问题存在性的证据,不是我们的方法近邻。)

**空白的精确形状**:不是"没人留痕",不是"没人测冲突",而是——**没有一个有原则的仲裁器,把'改变 / 冗余 / 例外'当作三种并列的记忆结局区分开;领域结构性地缺失'受保护例外'这第三种结局。**

## 二、人类记忆如何做到:图式记忆,以及它背后一个完整的认知模型

这一节是 SchemaMem 命名的由来,也是全文动机的认知根基。我们不堆砌零散原则,而是**以图式记忆为纲**,给出它在现代认知/系统神经科学里一个**完整、权威的实现模型**;§1.2 那个让所有工程系统翻车的输入,人脑用这个模型天然处理对了。

### 2.1 纲:图式是关于世界的生成模型(Bartlett → Piaget → 预测加工)

图式(schema)自 Bartlett(1932)起就被理解为**结构化的解释系统**:回忆不是逐字重放,而是**从图式出发的重建**。Piaget 把它做成动态的:新经验先被**同化**(assimilation)进既有图式;只有图式**反复**失配、累积出持续的失衡(disequilibrium),才驱动**顺应**(accommodation)式的结构重构——关键是**持续**失配、而非任一单次异常。

现代认知科学把"图式如何驱动记忆"统一到一个计算原则下——**预测加工 / 预测误差**(Rescorla-Wagner 1972;Friston 2010;Clark 2013):大脑是一个分层的**生成模型**,持续对下一个输入生成期望,而学习信号就是**预测误差**(观测与期望之差)。**图式 = 生成模型,预测误差 = 学习信号**——这不是我们的比喻,而是认知科学既有的框架(预测误差是人类认知的一等概念,与任何工程系统是否引用过它无关)。

### 2.2 体:CLS + SLIMM——大脑如何按预测误差把证据路由到不同去处

图式理论的**系统神经科学实现**,是**互补学习系统(CLS)**与其图式扩展 **SLIMM**。这一个模型(而非三条离散原则)同时给出了我们需要的全部结构:

- **两个互补存储(CLS;McClelland 1995,Kumaran 2016)**:海马是**稀疏、模式分离**的快系统,逐条存**具体情节**;新皮层是**分布式、重叠**的慢系统,跨情节抽取**潜在规律**(= 图式)。海马记忆通过**回放**逐步固化进新皮层,且回放可被**新颖性**调制。
- **按预测误差路由(SLIMM;van Kesteren 2012)**:mPFC 检测新输入与既有图式的"共振"(= 预测误差的反面)。
  - **共振高(一致 / 低预测误差)** → mPFC 触发快速新皮层整合,并**抑制海马**,与图式无关的细节**不被编码**。
  - **不一致(高预测误差)** → 触发海马 / MTL 去**单独编码**这个事件,连同上下文细节。

> **诚实边界(措辞须守到论文)**:CLS/SLIMM 是一个仍在积累证据的模型,后续复现工作对细节存在争议。我们用 CLS/SLIMM 作为**设计类比与机制蓝图**,不声称它是已盖棺的人脑事实;论文里以"在 CLS/SLIMM 视角下"限定,避免过度声称。此段引用留给正文时再核。

**这套模型天然产出四种去处,恰好对应 §1.2 该有的四种动作:**

| 输入 | 大脑做什么(CLS/SLIMM) | 记忆结局 |
|---|---|---|
| 一致、低预测误差 | mPFC 快速整合,抑制海马、不存无关细节 | **同化 + 压缩遗忘** |
| 高预测误差、**反复出现** | 海马先存,**回放反复**→ 固化进新皮层图式 | **信念重构** |
| 高预测误差、**孤立** | 海马**模式分离**单独留痕,**不**并入图式 | **受保护例外** |
| 检索时 | 新皮层**模式补全**从图式重建细节 | **重建式回忆** |

### 2.3 这个模型直接给出三个我们要用的推论

1. **改变需要"持续失配"**(Piaget 顺应 + CLS 回放固化):单次矛盾不足以改图式,要等违反反复出现 → 正当化**推迟提交 + 累积触发**。
2. **例外因不可被图式重建而必须逐字单独留**(SLIMM 海马路由 + CLS 模式分离):这也正是 von Restorff / 不一致项记忆优势的机制根源(需诚实:该效应**条件依赖、近似 U 形**,措辞用"优先编码并留存",非"意外的总被记得更牢")。
3. **能被图式模式补全的才可丢,不能补全的必须留**(CLS 新皮层模式补全 = Bartlett 重建):遗忘与例外保留是**同一条重建判据的两端**。

> **一个独立的工程旁证**:受控消融显示"逐字块打败结构化制品"(连知识更新类结构化都吃亏)。这印证推论 2 与 3——**例外必须逐字留**,不能被结构化抽取平滑掉。SchemaMem 因此是"图式做仲裁 + 逐字留例外"(= 新皮层图式 + 海马例外)的**双存储混合**,不是纯结构化,这条在论文主动接招。

## 三、我们的动机:为什么是现在、为什么是这个切法

1. **问题真实、高频、且是长期记忆最要命的错误来源**:改变 vs 例外的误判,每一次长期交互都在发生;综述亲口把它的三个后果(平滑例外、删长尾、盲目覆写)都列为未解风险。
2. **空白刚被一手来源确认还空着**:LoCoMo-Plus(2026-02)明确声明不碰信念修订;MAB 的冲突消解任务全场惨败(最强系统也仅六成),headroom 就在**标准基准**上。
3. **一个统一视角能收编三个分头判据**:与其再发明第四个启发式,不如指出这三条线共享同一个缺陷,并用**同一把尺子**同时修好它们——这才是"完整的、演化的动态模型",而不是三个技巧的拼盘。
4. **人类记忆给出现成蓝图**:§2 的三条原则不是装饰,而是分别对应我们三种结局的设计依据,让机制"有价值来源"而非"凭空设定"。

**动机一句话**:记忆演化的真问题不是"记不记得住旧值",而是"**面对一条矛盾证据,该不该现在就改信念**";领域缺的不是留痕能力,而是**一个把改变、冗余、例外区分开的有原则仲裁器**,以及它缺失的第三种结局——**受保护例外**。

## 四、核心心智模型:图式作为生成模型,残差作为唯一货币

### 4.1 图式是什么,它的价值是什么

**图式 = 关于一个实体的生成模型。** 对实体 e,图式 S_e 是一组**具名槽位**(如 diet、location),每个槽位挂一本**证据账本**(每条记录带违反度与时间)。图式能对"关于 e 的下一条观测"给出**期望**。

> **两条措辞纪律(引文核实后定,详见 `full_paper_zh.md` §4.1 与附录)**:
> 1. **"图式"专指信念层。** 逐字账本与受保护例外**不是图式的一部分**,而是其互补存储。因此只写"图式**裁定**例外并**路由**到受保护存储",绝不写"图式**保存**例外"——Ghosh & Gilboa 的必要特征之一正是"缺乏单元细节",而 Gilboa & Marlatte(Box 1)把容纳例外归给**范畴**而非图式。分层讲清则两处皆自洽,含混则自相矛盾。
> 2. **槽位有认知对应物**:"把输入匹配进被激活图式的变量槽"即 Gilboa & Marlatte 定义的**图式实例化(instantiation)**,正是 L2 在做的事。槽位结构不是工程便利。(注意:instantiation **不**支持"反复违反才重构",早期草稿曾误引。)

**图式的价值,不是"能区分改变和例外"这个功能,而是它作为生成模型为智能体买到的三样东西——且三者用同一个量度量:**

| 价值面 | 含义 | 度量 |
|---|---|---|
| **压缩** | 图式蕴含的观测无需逐字存 | 描述长度下降 |
| **预测** | 图式让"下一条观测"可预期 | 预测误差下降 |
| **稳定** | 图式让信念不被单次噪声推翻 | 越过阈值才修订 |

三样价值的共同度量,就是图式对每条新证据的**预测残差 r(x | S)**:x 有多"出乎图式预料"。

### 4.2 一个判据,一把尺子的两端

残差 r 是贯穿记忆全生命周期的**单一货币**。一条新证据进来,只问 r 和"是否复现":

| 残差 r | 复现? | 证据性质 | 图式动作 | 兑现的价值 |
|---|---|---|---|---|
| ≈ 0 | — | 冗余,图式已蕴含 | **遗忘原文** | 压缩 |
| 高 | 跨多个独立片段累积 | 图式系统性错了 | **重构期望**(旧值转为"已被取代") | 更新 |
| 高 | 孤立、不复现 | 真实但不可重建 | **保护为例外**(信念不动,逐字留) | 稳定 |

**离散标签,分级阈值。** 图式文献明说契合度是连续的("事件从不与既有图式完全一致或完全不一致",Gilboa & Marlatte p.624),而实现上 r 取三档离散值——**这不是矛盾,是分工:离散的是标签,分级的是阈值**。单条观测的连续残差由 LLM 打分,不可靠也不可复现,故离散化以保住可识别性;分级性在 MDL 阈值 k\*=⌈S/r̄⌉ 上恢复(违反越强,所需复现越少)。这是"你声称以图式为根基却把图式说是连续的量二值化了"这一质疑的标准答复。

**这是同一个测试的三个出口**:能被图式重建(r≈0)的就丢;不能重建又反复出现的,说明图式该改;不能重建又只出现一次的,就是例外。**遗忘和例外保留由此结构耦合**:凡不能被图式重建的,**永远进不了遗忘队列**——这一条同时堵上综述点名的两个 open risk(巩固抹例外、LRU 删长尾)。

### 4.3 完整的动态:五动作转移函数(每个动作都有 CLS/SLIMM 对应,非凭空提出)

记忆演化不是三个 if-else,而是一个由 `(残差 r, 累积计数 n)` 驱动的转移系统——**它是 §2.2 那张 CLS/SLIMM 路由表的算法化**。对每条新片段 x,图式状态按下列规则演化,每个动作标注其认知模型出处:

1. **同化(Assimilate)** — r≈0:x 与图式一致,不新增结构,原文可释放。
   ↔ *SLIMM:共振高 → mPFC 快速整合、抑制海马、不编码无关细节(压缩)。*
2. **累积(Accumulate)** — r 高但 n 未达阈值:登记违反、更新计数,**暂不改信念**。
   ↔ *CLS:高预测误差事件先存海马,等待回放;Piaget:失衡需持续累积。*
3. **重构(Accommodate)** — r 高且 n 越过阈值:改写期望为能同时解释旧证据与新违反的新值,旧值挂 supersedes 链。
   ↔ *CLS:海马反复回放 → 固化进新皮层图式;Piaget:顺应式结构重构。(对应 LongMemEval 知识更新类的直接解法。)*
4. **保护(Protect)** — r 高但违反孤立、始终不复现:登记为**受保护例外**,信念不动,**永不并入期望、永不进入重建、永不被遗忘**。
   ↔ *SLIMM:不一致事件由海马/MTL 单独编码;CLS:模式分离逐条留痕,不并入图式。*
5. **消解(Dissolve)** — 边界态:图式长期无支撑证据 / 全部槽位被例外淹没 → 图式失效回收。
   ↔ *图式在证据基被抽空后不再是有效生成模型。*

**"形成(Form)"是另一个边界态**:实体首次出现、无图式时,查询上下文为空,系统**退化为纯 RAG**——这在机制上保证"单跳事实类与检索基线走同一条路径",单跳持平不是巧合而是设计,可作证伪检验的"该平"侧。

> 五个动作 + 两个边界态,不是我们枚举出来的一组启发式,而是**一个生成模型在预测误差驱动下演化时的完整状态空间**,并逐一对应 CLS/SLIMM 的神经机制。这就是用户要求的"完整、演化的动态模型"。

### 4.4 一句话定位(投稿级)

> **我们把智能体的实体记忆建成一个图式(生成模型),用它对新证据的预测残差作为单一判据,把遗忘、信念更新、例外保留统一为一次三路仲裁,并引入现有全景结构性缺失的第三种记忆结局——受保护例外。** 预测误差是人类认知的一等概念(Rescorla-Wagner/Friston),我们直接采用;承重的新颖性在于把它落成 agent memory 里"改变/冗余/例外"的三路路由 + 双存储(图式 + 逐字例外)架构,而非发明一个新信号。

## 五、贡献(据可辩护度排序)

1. **改变 vs 冗余 vs 例外的三路仲裁**:以图式残差 + 复现统计为单一判据,给出各自的形式化规则与消融;引入"受保护例外"这一领域缺失的第三结局。
2. **重建门控的遗忘**:原文仅当图式能在容差 ε 内重建它时才被释放;给出可测 ε 与"删除后召回不降"的验证;并证明例外因不可重建而结构性免于遗忘。**实现上 `enable_forgetting` 与 `online_decay` 默认关闭,主表跑简单流末版本;开启后作为消融项报告**。
3. **与检索基线严格同构的底座**:无图式实体退化为纯 RAG,把增益干净归因于判据机制,以单跳持平作内建证伪检验。
4. **混合流诊断**:在 LoCoMo/LongMemEval 现成对话上做受控扰动(注入一次性例外 + 真改变),量出"最新即胜"系统在例外上的必然崩溃——这是标准基准结构上测不到的轴。

## 六、待办与风险(诚实上报)

- **ε 稳定性是整栋楼的地基**(用户已理解并认同):投稿前需用小样本验证残差判据能否稳定分开"改变"与"例外"。
- **基线由并行 session 承担**:Mem0 / A-MEM / MemoryBank 三条演化分支的对照跑由 A-line session 单独接线,本 session 专注 SchemaMem 侧机制与接入。当前实验状态见 `docs/eval/evolution_comparison_plan.md`(实验矩阵、阶段排序、handoff 契约的活文档)。
- **基准适配风险**:LongMemEval 知识更新类奖励覆写,首次覆写的消融(L3)在该类上可能不输完整系统(L4)——命门检查应改挂到自建的 drift-vs-anomaly 探针,LongMemEval 只证"整体不掉分"。
- **结果强度**:全上下文基线在 LongMemEval_S 上很强,诚实目标是"competitive + 增益集中于改变/例外类",非"全面超越"。**只关注准确率(acc),不把 token 效率作为卖点**(省 token 需额外实验,暂不纳入)。
- **撞车监控**:camera-ready 前重跑 arXiv 检索(PREMem/PISA/CAM/SleepGate 的后续引用),"例外留存 + 重建门控"是可能被抢的组合。
